Цель лабораторной работы
Целью лабораторной работы является изучение принципов разработки мобильных приложений для операционной системы Android, а также освоение работы с аппаратными датчиками устройства. В рамках работы требуется реализовать приложение для подсчета шагов пользователя.
Задание
Создать мобильное приложение под управлением ОС Android с использованием среды разработки Android Studio. Приложение должно использовать датчик акселерометра для определения шагов пользователя и отображать результат на экране.
